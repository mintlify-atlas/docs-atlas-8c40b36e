## Welcome to fluxer.py

**fluxer.py** is a powerful Python library for building bots and automated clients on the [Fluxer](https://fluxer.app) platform. It provides a clean, type-safe, and event-driven architecture that makes bot development straightforward and enjoyable.

## Quick Example

Here's a simple bot that responds to a ping command:

```python
import fluxer

bot = fluxer.Bot(command_prefix="!", intents=fluxer.Intents.default())

@bot.event
async def on_ready():
    print(f"Bot is ready! Logged in as {bot.user.username}")

@bot.command()
async def ping(ctx):
    await ctx.reply("Pong!")

if __name__ == "__main__":
    TOKEN = "your_bot_token"
    bot.run(TOKEN)
```

## Architecture

fluxer.py is designed with a clear separation of concerns:

- **HTTPClient** — Handles REST requests with automatic rate limiting
- **Gateway** — Manages WebSocket connection and event dispatch
- **Client** — Base event-driven interface for full control
- **Bot** — High-level command framework built on Client
- **Cog** — Modular system for organizing commands and events

## Requirements

- Python 3.10 or higher
- `aiohttp` for async HTTP requests
- `emoji` for emoji handling
- Optional: `livekit` for voice support
