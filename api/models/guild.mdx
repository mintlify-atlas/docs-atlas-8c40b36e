## Overview

The `Guild` class represents a Fluxer guild (also known as a server or community). It provides methods for managing members, roles, and emojis.

## Attributes

<ResponseField name="id" type="int" required>
  The guild's unique ID (snowflake).
</ResponseField>

<ResponseField name="name" type="str | None">
  The guild's name.
</ResponseField>

<ResponseField name="icon" type="str | None">
  The hash of the guild's icon.
</ResponseField>

<ResponseField name="owner_id" type="int | None">
  The ID of the guild owner.
</ResponseField>

<ResponseField name="member_count" type="int | None">
  The approximate number of members in the guild.
</ResponseField>

<ResponseField name="unavailable" type="bool">
  Whether the guild is unavailable due to an outage.
</ResponseField>

## Properties

<ResponseField name="created_at" type="datetime">
  When this guild was created (derived from snowflake ID).
</ResponseField>

<ResponseField name="icon_url" type="str | None">
  URL for the guild's icon, or None if the guild has no icon.
</ResponseField>

### ban

```python
async def ban(
    self,
    user_id: int,
    *,
    delete_message_days: int = 0,
    delete_message_seconds: int = 0,
    reason: str | None = None,
) -> None
```

Ban a user from this guild.

<ParamField path="user_id" type="int" required>
  User ID to ban.
</ParamField>

<ParamField path="delete_message_days" type="int">
  Number of days to delete messages for (0-7, default 0).
</ParamField>

<ParamField path="delete_message_seconds" type="int">
  Number of seconds to delete messages for (0-604800, default 0).
</ParamField>

<ParamField path="reason" type="str | None">
  Reason for audit log.
</ParamField>

**Example:**

```python
# Ban a user and delete their messages from the last day
await guild.ban(
    user_id,
    delete_message_days=1,
    reason="Spam"
)
```
