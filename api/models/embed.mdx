## Overview

The `Embed` class is a builder for creating rich embed objects that can be sent in messages. Embeds allow you to send richly formatted messages with titles, descriptions, fields, images, and more.

## Attributes

<ResponseField name="title" type="str | None">
  The embed title.
</ResponseField>

<ResponseField name="description" type="str | None">
  The embed description.
</ResponseField>

<ResponseField name="url" type="str | None">
  URL that the title links to.
</ResponseField>

<ResponseField name="color" type="int | None">

## Usage Example

```python
from fluxer import Embed

# Create a rich embed
embed = Embed(
    title="Hello, World!",
    description="This is a rich embed",
    color=0x5865F2
)

# Add fields
embed.add_field(name="Field 1", value="Value 1", inline=True)
embed.add_field(name="Field 2", value="Value 2", inline=True)

# Set footer and image
embed.set_footer(text="Footer text")
embed.set_image(url="https://example.com/image.png")

# Send the embed
await channel.send(embed=embed)
```

## Color Examples

Colors are specified as hexadecimal integers:

```python
# Common colors
red = 0xFF0000
green = 0x00FF00
blue = 0x0000FF
fluxer_purple = 0x5865F2

embed = Embed(title="Colored Embed", color=fluxer_purple)
```
