## Overview

The `Message` class represents a message sent in a Fluxer channel. It provides methods for sending, editing, deleting, and reacting to messages.

## Attributes

<ResponseField name="id" type="int" required>
  The message's unique ID (snowflake).
</ResponseField>

<ResponseField name="channel_id" type="int" required>
  The ID of the channel this message was sent in.
</ResponseField>

<ResponseField name="content" type="str" required>
  The text content of the message.
</ResponseField>

<ResponseField name="author" type="User" required>
  The user who sent this message.
</ResponseField>

<ResponseField name="timestamp" type="str" required>
  ISO 8601 timestamp when the message was created.
</ResponseField>

<ResponseField name="edited_timestamp" type="str | None">
  ISO 8601 timestamp when the message was last edited, or None if never edited.
</ResponseField>

<ResponseField name="guild_id" type="int | None">
  The ID of the guild this message was sent in, or None for DMs.
</ResponseField>

<ResponseField name="embeds" type="list[dict[str, Any]]">
  List of embed objects attached to this message.
</ResponseField>

<ResponseField name="attachments" type="list[Attachment]">
  List of file attachments on this message.
</ResponseField>

<ResponseField name="mentions" type="list[User]">
  List of users mentioned in this message.
</ResponseField>

<ResponseField name="pinned" type="bool">
  Whether this message is pinned.
</ResponseField>

<ResponseField name="reactions" type="list[Reaction]">
  List of reactions added to this message.
</ResponseField>

<ResponseField name="referenced_message" type="Message | None">
  The message being replied to, or None if this is not a reply.
</ResponseField>

## Properties

<ResponseField name="created_at" type="datetime">

<ResponseField name="channel" type="Channel | None">

<ResponseField name="guild" type="Guild | None">
  The guild this message was sent in (if cached).
</ResponseField>
