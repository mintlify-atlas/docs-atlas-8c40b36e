## Overview

The `HTTPClient` class provides an async HTTP client for interacting with the Fluxer REST API. It handles rate limiting, retries, error mapping, and authentication.

## HTTPClient

### Constructor

```python
from fluxer.http import HTTPClient

http = HTTPClient(
    token="your_token",
    is_bot=True,
    api_url="https://api.fluxer.app/v1",
    max_retries=4,
    retry_forever=False
)
```

#### Parameters

<ParamField path="token" type="str" required>
  Bot or user token
</ParamField>

<ParamField path="is_bot" type="bool" default="True">
  Whether this is a bot token (adds "Bot" prefix to auth header)
</ParamField>

<ParamField path="api_url" type="str" default="https://api.fluxer.app/v1">
  Base URL for the API. Use this to connect to self-hosted Fluxer instances.
</ParamField>

<ParamField path="max_retries" type="int" default="4">
  Maximum number of retries for failed requests
</ParamField>

<ParamField path="retry_forever" type="bool" default="False">
  Whether to retry requests indefinitely
</ParamField>

### Core Methods

#### request()

```python
await http.request(route, json=None, data=None, params=None, reason=None)
```

Make an authenticated request to the Fluxer API. Handles rate limiting, retries on 429/5xx, connection errors, and error mapping.

<ParamField path="route" type="Route" required>
  The API route to request
</ParamField>

<ParamField path="json" type="Any">
  JSON payload to send
</ParamField>

<ParamField path="data" type="aiohttp.FormData | None">
  Form data to send (for file uploads)
</ParamField>

<ParamField path="params" type="dict[str, Any] | None">
  Query parameters
</ParamField>

<ParamField path="reason" type="str | None">
  Reason for audit log (sent in X-Audit-Log-Reason header)
</ParamField>

<ResponseField name="return" type="Any">
  Parsed JSON response, or None for 204 No Content
</ResponseField>

#### close()

```python
await http.close()
```

Close the underlying HTTP session.

### Context Manager

## Route

Represents an API route. Used for rate limit bucketing.

### Constructor

```python
from fluxer.http import Route

route = Route(
    "GET",
    "/channels/{channel_id}/messages",
    channel_id="123",
    base_url="https://api.fluxer.app/v1"
)
```

#### Parameters

<ParamField path="method" type="str" required>
  HTTP method (GET, POST, PATCH, DELETE, etc.)
</ParamField>

<ParamField path="path" type="str" required>
  API path with placeholders (e.g., "/channels/{channel_id}/messages")
</ParamField>

<ParamField path="base_url" type="str" default="https://api.fluxer.app/v1">
  Base URL for the API
</ParamField>

<ParamField path="**params" type="Any">
  URL parameters to substitute into the path
</ParamField>

### Properties

- `url` - Full URL with parameters substituted
- `bucket` - Rate limit bucket key

## Example Usage

```python
from fluxer.http import HTTPClient, Route

async with HTTPClient("your_token") as http:
    # Get current user
    user = await http.get_current_user()
    print(f"Logged in as {user['username']}")
    
    # Send a message
    message = await http.send_message(
        channel_id="123456789",
        content="Hello, world!"
    )
    
    # Custom route
    route = Route("GET", "/users/@me")
    data = await http.request(route)
```
